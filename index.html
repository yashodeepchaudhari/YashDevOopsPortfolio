GymPro — AI-Powered Gym Management System

Jan 2025 – Mar 2026

End-to-end Django platform integrating three Gemini AI surfaces and two scikit-learn ML models behind a JWT-secured REST API. Shipped from a blank repository to a live AWS deployment with 32 automated tests and offline AI stubs.

  • AI 1 — Fitness Plan Generator: Structured-output Gemini pipeline (response_mime_type='application/json') returns a validated 7-day workout + diet plan; deterministic mock fallback for offline use.
  • AI 2 — FitBot Chatbot: Multi-turn Gemini chat with a dynamic system prompt injecting BMI, current plan, goal, and expiry; replays last 10 persisted messages as Gemini history.
  • AI 3 — Body Analyser: Gemini Vision returns a 3-section structured response (posture, focus, encouragement) with responsible-AI guardrails.
  • ML 1 — Churn Predictor: RandomForestClassifier on 8 behavioural features, stratified split, joblib persistence, surfaced as a ranked admin risk table.
  • ML 2 — Demand Forecaster: Seasonality-aware LinearRegression with Fourier-encoded time features (sin/cos on a 12-month cycle); 3-month forward projection on a Chart.js widget.
  • Hardening: JWT (1h access / 7d refresh), role-scoped querysets, Swagger/ReDoc via drf-spectacular, QR-code attendance, PDF receipts via reportlab.
  • Testing & Deploy: 32 pytest-django tests; AI tests patch the Gemini SDK via unittest.mock — runs offline and cost-free in CI. Deployed on AWS EC2 + RDS + S3 with gunicorn + WhiteNoise.
